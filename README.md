# avinashladdha.github.io

This website is built with [Jekyll](https://jekyllrb.com/) and uses the [Just the Docs](https://just-the-docs.github.io/just-the-docs/) theme.

## Local Development

1. Install Ruby and Bundler.
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Build and serve the site:
   ```bash
   bundle exec jekyll serve
   ```
4. Open http://localhost:4000 in your browser.

## Deployment

This site is configured to be deployed via GitHub Pages. Push changes to the repository to trigger a build.
